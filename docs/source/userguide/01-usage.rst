Tool usage guide
================

This Probe designer Pipeline provides a web UI, CLI-friendly endpoints, and a 
fully deployable source code base so you can run probe design jobs locally or 
integrate the service into automated workflows. Use the web interface for 
interactive use and monitoring, the CLI for automation, or deploy 
the source for full control and customization.
